# Celvion Stack website

Official static website for Celvion Stack, an on-device RAW photo stacking, AI denoise, super-resolution, editing, and export app for iPhone and iPad.

The site contains:

- Product homepage
- Support and troubleshooting
- Privacy policy
- Terms of use
- Open-source software and model licenses

The site uses plain HTML, CSS, and JavaScript with no analytics, cookies, third-party fonts, or build step. It is designed for GitHub Pages and supports English and Simplified Chinese.

## Local preview

Serve this directory with any static HTTP server and open `index.html`. A local server is required to preview the dynamically loaded plain-text license files.
